# 🌍 ClimAlte

An AI-powered tool that automatically generates TCFD-aligned 
climate risk long-lists for any company by searching peer 
disclosures, CDP reports, and industry frameworks.

## What it does
- Takes a company name and sector as input
- Searches real TCFD reports, CDP disclosures, and industry frameworks
- Categorises risks as Physical or Transition using TCFD methodology
- Rates materiality (High/Medium/Low) and timeframes
- Generates a structured long-list ready for client reporting
- Results downloadable as a text file

## Built with
- Python
- LangChain + LangGraph
- GPT-4o (OpenAI)
- Tavily Search
- Streamlit
